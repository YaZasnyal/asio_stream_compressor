asio_stream_compressor::compressor_statistics class
#include <compressor_statistics.h>

The compressor_statistics class holds stats.

Example:

auto& stats = compressor.get_statistics();
fmt::print("Total bytes written: {}\n",
           stats.tx_bytes_total.load(std::memory_order_relaxed));

Public types

struct value_slice
The value_slice struct stores stats right before they are reset.
using value_type = size_t
using stat_type = std::atomic<value_type>

Public functions

auto reset() -> value_slice noexcept
reset - resets all counters back to zero and returns all values that were stored in the compressor_statistics object.

Public variables

stat_type tx_bytes_total
tx_bytes_total - total number of bytes passed to the async_write_some() method.
stat_type tx_bytes_compressed
tx_bytes_compressed - total number of bytes sent to to the underlying stream (usually socket) through async_write_some()
stat_type rx_bytes_total
rx_bytes_total - total number of bytes written to the buffers that were provided to the async_read_some() method.
stat_type rx_bytes_compressed
rx_bytes_compressed - total number of bytes received from the underlying stream (usually socket).

Variable documentation

stat_type asio_stream_compressor::compressor_statistics::rx_bytes_total
